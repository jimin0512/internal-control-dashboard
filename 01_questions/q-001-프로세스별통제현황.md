# 프로세스별 통제 수와 Key 통제 비율은 어떻게 되는가?

## 질문

프로세스별 통제 수와 Key 통제 비율은 어떻게 되는가?

## 질문 목적

내부회계 전체 통제 현황을 파악한다. 이후 모든 설계평가·운영평가·미비점 분석의 기준이 되는 프로세스·통제 축을 정리하는 출발점 질문이다.

## 필요한 데이터

- [[schema-ic_process_master]]
- [[schema-ic_control_master]]

## 계산 또는 분석 방법

`ic_control_master.csv`를 프로세스명 기준으로 그룹핑하여 프로세스별 통제 수, Key여부="Y" 건수, 신규통제여부="Y" 건수를 집계한다. Key 비율은 프로세스별 Key 통제 수를 프로세스별 통제 수로 나누어 계산한다. 통제유형·통제방법·수행주기 분포는 전체 통제(180건)를 대상으로 각 컬럼의 값별 건수를 집계한다.
